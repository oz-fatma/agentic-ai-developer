# Senaryo 8: Connecting to a Remote

## Öğrenilenler
- git remote add origin <url>: local repoyu bir remote'a (GitHub gibi) baglar
- git remote -v: bagli remote'lari listeler
- origin: git clone yapildiginda otomatik verilen remote ismi
- Bizim ana repo (agentic-ai-developer) git clone ile olusturuldugu icin
  origin remote'u zaten otomatik olarak baglanmisti
- Sifirdan git init edilen bir repoda remote bulunmaz, git remote add ile
  manuel eklenmesi gerekir
