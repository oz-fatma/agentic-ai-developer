# Senaryo 2: Staging & Committing

## Öğrenilenler
- git add: değişiklikleri staging area'ya ekler
- git commit: staging area'daki değişiklikleri repository geçmişine kaydeder
- Staging area: commit öncesi değişiklikleri hazırladığın ara katman
- Commit: projenin belirli bir andaki anlık görüntüsü (snapshot)
