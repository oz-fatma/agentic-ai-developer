# Senaryo 1: Repository Başlatma

## Görev
Bu repo GitHub üzerinde oluşturulup git clone ile local'e indirildi.
git clone komutu, hedef klasörde otomatik olarak git init yapar,
bu yüzden ayrıca git init çalıştırmaya gerek kalmadı.

## Öğrenilenler
- git init: mevcut klasörde yeni bir Git repository oluşturur.
- git clone: uzak bir repoyu local'e indirirken bu init işlemini
  otomatik olarak kendi içinde yapar.
- .git klasörü, Git'in tüm proje geçmişini ve metadata'yı sakladığı yerdir.
