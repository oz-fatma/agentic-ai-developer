# Senaryo 3: Status & History

## Öğrenilenler
- git status: working directory ve staging area'nın durumunu gösterir
  (değişen, staged, untracked dosyalar)
- git log: commit geçmişini gösterir
- git log --oneline: her commit'i tek satırda özetler
- git log --oneline --graph: commit geçmişini ağaç yapısında gösterir
